# birth-rate-in-the-city-of-Lipetsk
ЛПЦ24 ОБ ПМ1
Тема проекта: Анализ динамики и прогнозирование уровня рождаемости в городе Липецк на основе социально-экономических и демографических данных.
Цель проекта: Построить модель, позволяющую прогнозировать коэффициент рождаемости в Липецке на среднесрочную перспективу, и выявить ключевые факторы, оказывающие на него наибольшее влияние.
Задачи проекта:
- Собрать и систематизировать статистические данные по рождаемости и потенциально влияющим факторам в Липецке за репрезентативный период 2000-2024 гг.
- Построить и сравнить несколько регрессионных моделей для прогнозирования уровня рождаемости.
- Проинтерпретировать результаты, сформулировать выводы и возможные рекомендации.
Аннотация:
Проект посвящен исследованию демографической ситуации в Липецке через показатели рождаемости. На основе открытых данных Липецкой области был сформирован датасет, включающий временные ряды ключевых показателей. В результате предобработки и анализа были выявлены устойчивые тренды и зависимости. С помощью методов машинного обучения построена прогнозная модель, показавшая хорошую точность.
